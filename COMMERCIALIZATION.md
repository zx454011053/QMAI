# 青幕AI写作（QMAI）商业化变现方案

**版本**：v1.0  
**日期**：2026年5月31日  
**现状**：软件刚发布，无存量用户，MIT开源，无大模型合作方  
**策略方向**：先快速现金流，后长期稳定收入

---

## 一、软件定位分析

### 1.1 软件类型
桌面端 AI 写作工具（Tauri 2 跨平台），面向长篇网文创作场景。

### 1.2 核心竞争力
| 维度 | 说明 |
|------|------|
| 记忆系统 | 章节摄取 + 混合检索，解决 AI 写长篇忘前文的核心痛点 |
| 审查系统 | 六维审稿 + 连贯性检查，防止人设崩坏/时间线混乱 |
| 图谱功能 | 人物关系/伏笔/事件可视化追踪 |
| 本地架构 | 数据全部本地存储，隐私安全 |
| 角色灵魂 | NvwaSKILL 框架，保证角色一致性 |

### 1.3 目标用户群体
- **核心用户**：网文日更作者（日产3000-5000字，需要保持更新频率）
- **潜力用户**：小说策划者、AI辅助写作爱好者
- **用户规模预估**：中国网文作者约 200 万+，日更活跃作者约 30-50 万

### 1.4 用户付费意愿判断
- 网文作者对"提效工具"付费意愿中等偏高（月收入从几百到几万不等）
- 对"能直接帮赚钱"的工具（如提升日更速度 → 多签约 → 多收入）付费动力强
- 价格敏感，但对确实有效的工具愿意付月费 20-50 元

## 三、第一阶段：用户获取与快速现金流（0-3个月）

### 3.1 网盘分发模式

**分发平台选择**：
- 蓝奏云（无需会员即可下载，网文圈常用）
- 百度网盘（用户基数大）
- 夸克网盘（年轻用户多）
- 123网盘（无限速，体验好）

**下载激励机制**：

```
用户下载软件 → 注册账号 → 获得 3 天高级功能体验
                          → 分享下载链接给 1 人 → 再获 3 天
                          → 老用户每日登录 → 获得积分（可兑换高级功能天数）
```

**具体实施**：
1. 软件内嵌一个轻量账号系统（本地验证 + 在线激活码模式）
2. 用户首次启动输入手机号或邮箱获取激活码
3. 新用户自动获得 3 天 Pro 体验
4. 激活码生成器部署在低成本云函数上（如腾讯云 SCF）

**预期效果**：
- 月下载量目标：1000-3000（前3个月）
- 注册转化率：60-70%（因为需要注册才能用高级功能）
- 成本：约 50-100 元/月（云函数 + 网盘存储）

### 3.2 社区推广渠道

**精准渠道（按优先级排序）**：

| 渠道 | 操作方式 | 预期效果 |
|------|----------|----------|
| 起点/番茄作者群 | QQ群/微信群发使用教程 | 直接触达目标用户 |
| 龙空论坛 | 发帖 + 连载使用日记 | 网文作者聚集地 |
| B站 | 录制使用教程视频（AI写小说实测） | 年轻写作者多 |
| 小红书 | 发"AI写作工具测评"笔记 | 女性写作者多 |
| 知乎 | 回答"网文写作工具推荐"类问题 | 高质量长尾流量 |
| V2EX/少数派 | 发产品介绍帖 | 技术向用户，可能贡献PR |
| 公众号/头条号 | "AI写网文月入XXX"类内容 | 吸引尝试者 |

**内容策略**：
- 核心卖点文案：「写到200万字，AI还能记住第1章的伏笔」
- 对比视频：普通ChatGPT写50章 vs 青幕AI写50章的效果对比
- 案例展示：用软件写一个10万字小说的完整过程记录

## 四、第二阶段：基础变现（3-6个月）

### 4.1 大模型服务推广佣金（快速现金流核心）

**模式说明**：

多数大模型平台（如 DeepSeek、智谱、百度千帆、阿里通义、硅基流动）都有推广计划或 API 返佣机制。在软件内引导用户通过你的推广链接注册并充值 API，你获得佣金。

**具体操作**：

1. **注册各平台推广计划**：
   - 硅基流动（SiliconFlow）：有明确的推广返佣，用户充值返10-20%
   - DeepSeek：申请 API 推广合作
   - 智谱AI（GLM）：开发者生态合作
   - 阿里通义/百度千帆：申请渠道合作

2. **软件内集成方式**：
   - 设置页面"模型配置"处，提供"一键注册"按钮（携带推广参数）
   - 新用户首次配置模型时，弹出推荐面板：「推荐使用 DeepSeek，性价比最高」
   - 提供"模型选择指南"文档，内嵌推广链接
   - **不强制**用户使用特定平台，只做推荐引导

3. **返现激励设计**：
   ```
   用户通过推广链接注册大模型平台：
   - 首次充值 ≥ 10元：用户获得 3 天 Pro 功能
   - 首次充值 ≥ 50元：用户获得 15 天 Pro 功能
   - 首次充值 ≥ 100元：用户获得 1 个月 Pro 功能
   ```

**预期收入**：
- 假设月活 1000 用户，30% 通过推广链接注册，人均充值 50 元/月
- 佣金比例 10-15%
- 月收入：1000 × 30% × 50 × 12% ≈ **1800 元/月**
- 用户增长后可线性放大

### 4.2 API 中转服务（高利润）

**模式说明**：

自建 API 中转站，统一对接多家大模型，用户在软件内直接使用你的 API 地址，按量计费。

**优势**：
- 用户无需自己注册多个平台
- 你赚取中间差价（批量采购价 vs 零售价）
- 可以提供"免配置"体验（开箱即用）

**定价策略**：
```
采购成本（以 DeepSeek V3 为例）：约 1元/百万token
零售定价：1.5-2元/百万token（加价50-100%）
用户感知：比直接买贵一点，但免配置、一站式、稳定
```

**技术实现**：
- 部署 one-api 或 new-api 开源中转项目
- 云服务器成本：约 100-200 元/月（轻量级即可）
- 用户在软件设置中选择"青幕云API"，输入购买的密钥即可

**预期收入**：
- 假设 500 活跃写作用户，人均日消耗 5 万 token
- 月消耗：500 × 50000 × 30 = 7.5 亿 token
- 利润：0.5元/百万token × 750 ≈ **375 元/月**（初期）
- 用户量上来后此项收入增长最快

### 4.4 启动广告（补充收入）

**广告形式选择**：

| 形式 | 适用性 | 用户干扰度 |
|------|--------|------------|
| 启动等待页（3秒可跳过） | ✅ 适合桌面软件 | 低 |
| 侧边栏推荐位 | ✅ 不影响写作区 | 极低 |
| 功能解锁前插页 | ⚠️ 需谨慎 | 中 |
| 信息流广告 | ❌ 不适合工具类 | — |

**推荐方案**：
- 免费版用户：启动时展示 3 秒推荐页（自有广告位，推广合作产品或大模型服务）
- 侧边栏底部：展示"推荐工具/课程/社区"链接
- Pro版：无任何广告

**广告来源**：
- 自营广告（推广大模型服务、写作课程、网文平台）
- CPA/CPS合作（写作相关产品的推广佣金）
- 暂不接入广告联盟（用户量不够，且桌面端广告联盟体验差）

**预期收入**：
- 初期以自营推广为主，收入已计入大模型佣金
- 后期可接写作培训课程CPS，单价 10-50 元/单

## 八、风险评估与应对

### 8.1 风险矩阵

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|----------|
| 用户增长缓慢 | 高 | 高 | 多渠道推广 + 内容营销 + 免费版足够好用 |
| 付费转化率低 | 中 | 高 | 免费/付费边界调优 + 限时优惠 + 试用期 |
| 大模型推广政策变化 | 中 | 中 | 多平台分散 + 发展自有API中转 |
| 竞品出现 | 中 | 中 | 持续迭代核心功能 + 社区运营 + 先发优势 |
| 用户破解绕过付费 | 低 | 低 | 核心验证放服务端 + 不过度防破解 |
| 支付/资质合规问题 | 低 | 高 | 使用正规聚合支付 + 必要时注册个体户 |

### 8.2 合规要求

1. **个人收款**：月收入 < 3万可用个人支付宝/微信收款；超过建议注册个体工商户
2. **广告合规**：推荐内容需标注"推广"字样，不做虚假宣传
3. **隐私保护**：
   - 不上传用户小说内容到自有服务器
   - 统计数据仅收集功能使用频次，不收集内容
   - 在设置中提供隐私政策说明
4. **开源协议**：MIT协议允许商业化，但建议将商业化代码（licensing模块）单独闭源
